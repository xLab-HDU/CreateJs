本项目是基于可交互的素材处理工具，可以直接运行html文件进入网页，整合了缩放、裁剪设定，可以按照自定义区域快速处理素材，但是压缩后图片质量效果有限。
网页操作方法：
1.导入一张图片作为参考；
2.设定缩放比例和裁剪区域，确认裁剪区域；
3.导入素材文件夹，开始裁剪
————————————————————————————————————————————————————————————————————————————————————————————————————————————————
⚠如果想要获取高质量裁剪及压缩图片，推荐使用PS批处理图片
PS操作方法如下：
1.导入一张图片作为参考；
2.点击“窗口-动作”打开动作记录界面；
3.在动作记录界面下方点击“+”新增动作，点击“⚪”切换为红色表示开始录制；
4.使用“裁剪工具(C)”调整裁剪区域，确认好区域后回车Enter；
5. Ctrl+Alt+I 修改压缩参数；
6.点击“文件-存储为”保存图片；
7.回到动作记录界面，点击“⬜”结束录制，完成动作录制；
8.点击“文件-自动-批处理”，界面左侧选择“源文件夹”即要处理的图片，界面右侧选择“目标文件夹”即输出位置，点击“确定开始批处理素材”；
